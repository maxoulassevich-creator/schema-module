# 1.0.11

- Производительность: проверка/миграция схемы БД (`Db::install()` → `isTableExists` + `SHOW COLUMNS`) теперь выполняется не более одного раза за запрос. Раньше она запускалась при каждом вызове `activeForKind()` / `activeForRequest()`, то есть на каждой странице сайта через событие `main:OnEndBufferContent` — это добавляло лишние служебные SQL-запросы к каждому хиту. Добавлен внутренний флаг, схема данных при этом не меняется.
