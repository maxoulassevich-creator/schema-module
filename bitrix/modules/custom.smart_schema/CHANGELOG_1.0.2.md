# Smart Schema Enterprise for Bitrix 1.0.2

Исправления по реальным HTML-страницам newrelease.relod.ru:

- Модуль теперь учитывает не только JSON-LD, но и существующую microdata-разметку `itemscope/itemtype`.
- Если в шаблоне уже есть корректная microdata `BreadcrumbList`, модуль не предлагает дублировать её в JSON-LD.
- Если карточка товара уже содержит JSON-LD `Product`, модуль не предлагает второй `Product`.
- Если список блога уже содержит JSON-LD `Blog` с `blogPost`, модуль не предлагает лишние `CollectionPage` и `ItemList`.
- Если детальная страница поста уже содержит microdata `Article`, модуль не предлагает дублировать её как `BlogPosting`.
- Для категорий каталога исправлено извлечение товаров: теперь берутся карточки `.catalog-block__item`, а не пункты меню шапки/футера.
- Для хлебных крошек исправлено извлечение последнего пункта через `link itemprop="item"` и исключены выпадающие пункты из dropdown.
- Для изображений исправлена обработка lazyload: теперь учитывается `data-src`, а не пустой `src="data:image/gif..."`.
- Добавлено автоматическое помечение устаревших pending-предложений как «не вносить» при новом сканировании.
