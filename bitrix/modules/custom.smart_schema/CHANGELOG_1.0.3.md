# Smart Schema Enterprise 1.0.3

## Добавлено

- Ручная проверка дополнительных страниц: поле `manual_urls`, один URL на строку.
- Предложения для ручных URL привязываются к точному URL, а не ко всем страницам типа.
- Реальная проверка после нажатия «Внести»: модуль загружает страницу-пример, ищет JSON-LD-блок модуля и сохраняет результат проверки в карточке предложения и журнале.
- Кнопка «Проверить» для повторной проверки уже применённого пункта.
- Режим «Заменить своей»: модуль перед вставкой своего JSON-LD удаляет старые JSON-LD-блоки такого же Schema.org-типа из HTML-буфера и отключает корневой `itemscope/itemtype` у microdata-блоков этого же типа.
- Метки `data-smart-schema-*` у выводимых JSON-LD-блоков для точной проверки, что разметку вывел именно модуль.
- Аудит теперь помечает некорректные, дублирующиеся и чрезмерно крупные Schema.org-блоки как кандидаты на замену.

## Важно

Замена выполняется в HTML-буфере вывода сайта и не переписывает файлы шаблона Аспро/Битрикс на диске.
