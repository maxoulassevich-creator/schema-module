# 1.0.9

- Исправлено определение типа страницы для FAQ-страниц на аккордеоне Аспро Премьер (например, `/help/faq/`). Раньше вопросы и ответы не извлекались, поэтому модуль выводил безопасный `WebPage` вместо `FAQPage`.
- Добавлен разбор Bootstrap-аккордеона: триггер-вопрос с `data-toggle="collapse"` / `data-bs-toggle="collapse"` / `aria-controls` и связанный по `id` скрытый блок с ответом объединяются в пару вопрос–ответ.
- FAQ теперь собирается из двух источников: аккордеон и прежний шаблон «заголовок-вопрос + следующий блок-ответ». Дубликаты вопросов удаляются.
- Ложные срабатывания исключены: в пару попадают только элементы, чей видимый текст оканчивается на «?» и у которых найден непустой блок ответа (более 20 символов). Навигационные якоря и служебные `collapse`-переключатели игнорируются.
- Поведение без данных не изменилось: если реальные пары вопрос–ответ на странице не найдены, модуль не выдумывает `FAQPage`, а оставляет безопасный `WebPage`.

После обновления до 1.0.9 очистите кеш Битрикса и запустите анализ заново, затем подтвердите пункт `FAQPage` для FAQ-страницы.
